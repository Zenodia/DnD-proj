# Chapter 04 — The Shadow Guild

## Purpose

Shift the fantasy from solo survival to command mastery. The player now wins by orchestrating specialized helpers.

## Story summary

After defeating an elite enemy in a near-hopeless raid, the player unlocks the ability to forge shadow companions. Each summon has a tactical identity: scout, defender, disruptor, finisher, archivist. The player starts building a private force that makes them look almost mythic to ordinary hunters.

## Learning goal

Agentic Harness + Produce.

## Scenes

### elite_fall
The player defeats a powerful named enemy.

### shadow_awakening
The first companion rises, terrifying everyone present.

### harness_trial
The player learns that multiple helpers need coordination rules.

### guild_strain
Rival hunters suspect the player is hiding illegal power.

### synchronized_clear
The player clears a multi-room dungeon by chaining companion roles.

## Choices

- Raise the strongest enemy first.
- Raise the smartest enemy first.
- Reveal the shadows publicly.
- Keep the new force secret.

## Win state

The player builds the first stable version of the Shadow Guild.

## Fail state

Poor coordination causes summons to waste resources or create unintended danger.

## Rewards

- Companion roster system
- Command combo mechanic
- New codex entries for each shadow role
