# Systems Map

## The 7-step pattern as gameplay

The user-provided framework defines a horizontal 7-step pattern: Trigger, Collect, Enrich, Reason, Produce, Persist, Notify.[cite:2] This should be visible in the game as an actual mission loop rather than a hidden curriculum.

| Step | In-world meaning | Gameplay expression |
|---|---|---|
| Trigger | A gate anomaly or mission alert appears | Alarm screen, map pulse, urgent briefing |
| Collect | Raw evidence is gathered | Dialogue choices, scans, relic inspection, NPC reports |
| Enrich | Data is cleaned and classified | Tagging clues, identifying enemy type, sorting signal from noise |
| Reason | The player chooses a plan | Tactical branch, team loadout, path selection |
| Produce | The plan becomes action | Combat phase, crafted item, squad command, ritual execution |
| Persist | Learning is stored | Journal entry, unlocked codex, pattern archive, saved tactics |
| Notify | Others are warned or aligned | Civilian alert, guild report, squad ping, world-state update |

## The 5 layers as progression disciplines

The same framework defines five vertical layers: Guardrail, Spec, TDD, Agentic Harness, and Vibe Coding.[cite:2] These should function as “schools of mastery” in the game.

| Layer | In-world translation | Gameplay function |
|---|---|---|
| Guardrail | Dungeon laws and forbidden actions | Prevents reckless choices, teaches safety constraints |
| Spec | Mission contracts and success criteria | Clarifies win states and branch conditions |
| TDD | Trial checks and proof rituals | Requires validation before major actions unlock |
| Agentic Harness | Shadow companion coordination | Enables multi-agent play, automation, and combo planning |
| Vibe Coding | Sandbox improvisation | Lets players attempt risky shortcuts or prototype tactics |

## Priority rule

The design priority should stay exactly as requested by the user: Guardrail-Driven Development over Spec-Driven Development over Test-Driven Development over Agentic Harness over Vibe Coding.[cite:2] In gameplay terms, this means a flashy improvised move should still fail if it violates mission law, and a well-coordinated team should still collapse if the objective was vague from the start.
